## 上传格式
1. 命名格式：每个文件按照“模块序号-内部序号-算法英文名称”命名。1-OCR模块，2-图像色彩修复模块，3-图像纹理修复模块，4-算法优化模块，5-新增需求模块。内部序号负责人完成排序即可。
2. 上传方式：建立分支上传，不要直接上传到main分支。
3. 上传内容：主要上传模型训练、测试涉及的代码，权重文件、数据集等不要上传git仓库，统一上传至百度网盘。
